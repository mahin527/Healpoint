# Healpoint – Healthcare & Medical Website Template

[![Live Demo](https://img.shields.io/badge/demo-live-brightgreen.svg)](https://healpoint-demo.netlify.app/)

Healpoint is a modern, clean, and fully responsive healthcare & medical website template built using pure HTML, CSS, and JavaScript. It is designed for clinics, hospitals, doctors, medical centers, and health-related startups who want a professional online presence without heavy frameworks.

## 🔗 Live Preview
Check out the live demo here:  https://healpoint-demo.netlify.app/

## ✨ Features
- **Medical-Focused UI/UX:** Clean, trustworthy, and patient-friendly design.
- **Fully Responsive Layout:** Works perfectly on mobile, tablet, and desktop.
- **Smooth Animations:** Subtle scroll animations for a premium feel.
- **Pure Vanilla Code:** No Bootstrap, no Tailwind, no frameworks.
- **SEO Friendly:** Semantic HTML structure for better search visibility.
- **Easy Customization:** Well-structured CSS with reusable variables.


## 📂 File Structure
```text
healpoint/
├── assets/
│   ├── css/        # Style sheets
│   ├── images/     # Optimized images and icons
│   └── js/         # JavaScript functionality
└── index.html      # Main HTML file
```

## 🚀 Getting Started

#### To use this template:

2. Extract the ZIP file.

3. Open index.html in your favorite web browser.

4. To customize, edit the CSS variables in assets/css/style.css.
